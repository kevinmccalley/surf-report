# Groundswell Blog — Author Guide for Laura

Welcome to the Groundswell blog! This guide walks you through everything you need to write and publish a surf article. No technical knowledge required.

---

## Getting to the Studio

1. Go to **[groundswell.surf/studio](https://groundswell.surf/studio)**
2. Sign in with your Groundswell account (same login as your regular Groundswell access)
3. You'll see the Sanity Studio — a clean content editor with three sections in the left sidebar: **Posts**, **Authors**, and **Categories**

> **Dev environment**: When working on the dev site, use **[dev.groundswell.surf/studio](https://dev.groundswell.surf/studio)**

---

## Writing a Post — Step by Step

### 1. Create a new post

- Click **Posts** in the left sidebar
- Click the **pencil/compose icon** (top right of the posts list) to start a new post
- You'll see a form split into three tabs: **Content**, **SEO & Meta**, and **Settings**

### 2. Fill in the Content tab

| Field | Required? | Tips |
|---|---|---|
| **Title** | ✅ Yes | Keep it punchy. Max 100 characters. Example: *"How to Read a Swell Forecast Like a Pro"* |
| **Slug** | ✅ Yes | Click "Generate" — it auto-fills from your title. Don't change it after publishing. |
| **Excerpt** | ✅ Yes | 1–2 sentences that tease the article. Shows in the blog grid and Google results. Max 300 chars. |
| **Cover image** | ✅ Yes | Drag and drop or click to upload. Best size: **1200 × 630 px**. Always add alt text. |
| **Author** | ✅ Yes | Select from the dropdown. (Add yourself as an author first — see below.) |
| **Categories** | Optional | Pick 1–3 that best describe the topic. |
| **Body** | ✅ Yes | This is where you write the article. |

### 3. Writing the body

The body editor looks like a simplified word processor. Here's what each button does:

- **Normal / H2 / H3 / H4** — Heading levels. Use H2 for main sections, H3 for sub-sections.
- **B / I / Code** — Bold, italic, inline code.
- **Link icon** — Highlight text, click this, paste the URL. Toggle "Open in new tab" for external links.

**Special blocks** (click the + icon at the start of a new line):
- 📷 **Image** — Insert an inline photo with alt text and optional caption
- 💻 **Code block** — For any technical content (shell commands, file examples). Set the language for syntax highlighting.
- 💡 **Callout** — Highlight important info as Info, Tip, Warning, or Important boxes

### 4. SEO & Meta tab (optional but recommended)

- **SEO title** — If you want a different title in Google search results than the post title. Max 60 chars. Leave blank to use the post title.
- **SEO description** — The snippet shown in Google results. Max 160 chars. Leave blank to use the excerpt.

### 5. Settings tab

- **Published at** — Set the date and time to publish. Leave blank to save as a draft.
  - ✅ Set a past date → publishes immediately
  - 📅 Set a future date → post goes live automatically at that time
  - ⬜ Leave blank → stays as draft, never visible on the site
- **Featured post** — Toggle this ON for the post you want to appear as the big hero card at the top of the blog page. Only one post shows as featured at a time (the most recently featured one wins).

### 6. Publish

- Click **Publish** (top right) when you're ready
- The blog updates within about **60 seconds** (no rebuild needed)
- Check it at **[groundswell.surf/blog](https://groundswell.surf/blog)**

## Adding Yourself as an Author

Before writing your first post, add yourself as an author:

1. Click **Authors** in the left sidebar
2. Click the compose icon to create a new author
3. Fill in: Name, Slug (auto-generate), Role (e.g. "Surf Writer"), Bio (optional), Avatar photo
4. Click **Publish**

## Image Guidelines

| Type | Recommended size | Notes |
|---|---|---|
| Cover image | 1200 × 630 px | Landscape. This is the OG card too. |
| Inline images | 900 × 600 px max | Any ratio works — cropped to 16:9 in the reader |
| Author avatar | 200 × 200 px | Square, face clearly visible |

**Alt text tips for surf photos:**
- ✅ *"Aerial view of Pipeline breaking on a six-foot swell, Oahu, Hawaii"*
- ✅ *"Close-up of a longboarder trimming on a clean chest-high wave"*
- ❌ *"wave photo"* — too vague
- ❌ *"IMG_4532.jpg"* — never use filenames

**Good free sources for surf imagery:**
- [Unsplash](https://unsplash.com/s/photos/surfing) — search "surfing", "waves", "ocean"
- [Pexels](https://www.pexels.com/search/surfing/)

## Editing After Publishing

- Open the post in Studio, make your changes, and click **Publish** again
- Changes go live within 60 seconds
- **Do not change the slug** after a post is published — it breaks the URL

---

## SEO Tips for Surf Content

1. **Put the main keyword in the title**: e.g., "10-Day Surf Forecast" not "Planning Your Week at the Beach"
2. **Lead with the key info**: Don't bury the point — search engines (and surfers) want answers fast
3. **Use H2 headings for main sections**: Makes the article easy to scan and improves SEO structure
4. **Link to related Groundswell pages**: e.g., link "surf forecast" to `groundswell.surf` in the body
5. **Write an excerpt that sells the click**: It shows in Google — make it specific and compelling
6. **Aim for 600–1500 words**: Shorter is fine for news/updates; longer for guides
